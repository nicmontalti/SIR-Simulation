# Building

La compilazione è gestita tramite CMake. Per compilare il codice è sufficiente creare una directory di build, generare il Makefile tramite CMake e compilare con make.

## Prerequisiti

 - CMake 3.16+
 - SFML 2.5+
 - ROOT

## Building and running
    mkdir build
    cd build
    cmake ..
    make

    ./SIR_simulation
